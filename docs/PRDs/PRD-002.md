<a id="prd-databucket-plugin-0001"></a>

## 1. Tổng quan

`prd:databucket-plugin-0001`

Tạo **DataBucketPlugin** — một bộ static wrapper class cho Databuckets Unity SDK v1.0.6. Plugin đóng gói toàn bộ 7 API của SDK vào một class duy nhất với các tính năng bảo vệ bổ sung (kiểm tra Init state, log error khi gọi sai thứ tự).

- **Đối tượng sử dụng:** Unity developers tích hợp Databuckets analytics
- **Nền tảng:** Unity (C#)
- **SDK gốc:** Databuckets Unity SDK v1.0.6
- **Version plugin:** 1.0.0
- **Namespace:** `DataBucketPlugin`

### Mục tiêu chính

1. Đơn giản hóa việc sử dụng Databuckets SDK qua static methods
2. Tự động kiểm tra Init state trước khi gọi bất kỳ API nào
3. Log Error rõ ràng khi developer gọi sai thứ tự
4. Cung cấp tài liệu README dễ hiểu, CHANGE_LOG theo version, và Sample script

## 3. Tính năng (Features)

### 3.1. Init Wrapper

`feature:wrapper-init-0001`
> Implements: [`prd:databucket-plugin-0001`](#prd-databucket-plugin-0001)

- **API Wrapper:** `DataBucketWrapper.Init(string apiEndpoint, string apiKey)`
- **Logic bổ sung:**
  - Kiểm tra nếu đã Init rồi → log warning và return (không gọi lại)
  - Set flag `_isInitialized = true` sau khi Init thành công
- **SDK gốc:** `DatabucketsTracker.Init(apiEndpoint, apiKey)`

<a id="feature-sample-script-0009"></a>

### 3.9. Sample Script

`feature:sample-script-0009`
> Implements: [`prd:databucket-plugin-0001`](#prd-databucket-plugin-0001)

- **File:** `samples/DataBucketWrapperSample.cs`
- **Kiểu:** MonoBehaviour gắn vào GameObject
- **Chức năng:** Chạy tất cả 7 API trong `Start()`:
  1. `Init()` — Khởi tạo SDK
  2. `EnableExceptionTracking()` — Bật exception tracking
  3. `SetCommonProperty()` — Set 1 property
  4. `SetCommonProperties()` — Set nhiều properties
  5. `Record()` — Ghi event đơn giản
  6. `RecordWithTiming()` — Ghi event với timing
  7. `ForceEndSession()` — Kết thúc session
  8. `DisableExceptionTracking()` — Tắt exception tracking
